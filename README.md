# Portfolio — gensosanso

A personal portfolio site designed with a desktop OS visual metaphor, with windows, a taskbar, and bilingual content.

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **React Context** for state management
- **Markdown** (gray-matter + react-markdown) for content

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

Content lives in `/content` as `.md` files with YAML frontmatter. Each section has both an English (`.en.md`) and Chinese (`.zh.md`) version. Adding a new entry is as simple as adding a new file.
